In this page I will post my experiments and projects that I'm working with.
Serverless Chat is a serverless app that was made using some AWS Services.

  • AWS API Gateway
  • AWS Simple Storage Service (S3)
  • AWS Lambda
  • AWS DynamoDB
  • AWS CloudFront

All of them are using only the AWS Free Tier therefore I'm not paying anything until I exceed this limit, you can see more information about it here. The objective of this project is to discover what can be done with the AWS Services without spending much money or any money at all.

To fulfill this project I'm using some programming languages, in the front-end I'm using a pre-made template made with HTML5, CSS and SASS, I did not make the template so I could focus only on the core of the project which is the back-end. At the back-end I'm using GO to develop all Lambda functions that I used to call other AWS services like DynamoDB and some others that I will implement in the future, also Javascript to call my Lambda functions and that's it. To host this web page I'm using a S3 bucket as a static host and CloudFront to certify and optimize my domain from Google Domains that I already had so I'm re-using it to save me some money.
